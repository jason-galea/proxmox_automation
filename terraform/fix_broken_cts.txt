If you "Sleep" while a container is running, it will break.
It will be stuck "starting" in the GUI.

To fix:
    From root@prox:
        Kill the "lxc-start" process:
            $ ps -aux | grep lxc
            $ kill -9 <PID>
        Kill "vzstart" process, EG "UPID:prox:000004DD:0000179C:61F7AD0E:vzstart:???:root@pam"
            $ ps -aux | grep ???
            # kill -9 <PID>
        Remove the CT config file
            $ rm /etc/pve/lxc/???.conf
        Remove lock file
            $ rm /run/lock/lxc/pve-config-???.lock
        Remove orphaned root volume:
            $ rm ????
            OR:
            WebUI --> local-lvm --> CT Volumes --> Remove


    From localhost:
        Remove all terraform files
            $ ls /mnt/c/Users/jg/Programming/proxmox_automation/terraform/terraform.tfstate*
            $ rm /mnt/c/Users/jg/Programming/proxmox_automation/terraform/terraform.tfstate*
